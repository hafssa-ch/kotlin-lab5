# Lab 5 – Récursion : Calculer la factorielle (Kotlin)

## Objectifs
À la fin de ce lab, l’étudiant est capable de :
- Comprendre le principe de la récursion
- Identifier un cas de base
- Implémenter la factorielle en Kotlin
  - avec une boucle (approche itérative)
  - avec une fonction récursive
- Suivre l’exécution d’une fonction récursive

## Contexte
La factorielle d’un entier positif n (notée n!) est définie comme suit :
- 0! = 1
- 1! = 1
- n! = n × (n − 1)! pour n > 1

Ce lab compare deux approches :
- itérative (boucle)
- récursive (appel de fonction sur elle-même)
